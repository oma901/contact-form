# Контактная форма на HTML, CSS, JavaScript и PHP

Форма отправляет данные на PHP без перезагрузки страницы.  
Проверка данных есть и в JavaScript, и в PHP. Ответ от сервера приходит в формате JSON.

## Как запустить проект
### Вариант 1. Через встроенный сервер PHP

Откройте терминал в папке проекта и выполните команду:

```bash
php -S 127.0.0.1:8000
```

После этого откройте в браузере:

```text
http://127.0.0.1:8000/
```
### Вариант 2. Через OSPanel

1. Поместите папку проекта в папку `domains` внутри OSPanel.
2. Запустите OSPanel.
3. Откройте сайт через локальный домен или адрес, который покажет OSPanel.

## Как использовать

1. Откройте страницу проекта в браузере.
2. Введите имя, email и сообщение.
3. Нажмите кнопку `Отправить`.
4. Если поля заполнены неправильно, ошибки появятся под нужными полями.
5. Если все правильно, появится сообщение: `Форма успешно отправлена`.

## Что проверяется
- имя не должно быть пустым;
- email должен быть похож на настоящий email;
- сообщение не должно быть пустым.
